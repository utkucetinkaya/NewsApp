# NewsApp

## About ##
NewsApp is an application that lists current news. Users can search for news in the search bar. Goes to the news site via WebKit to access news details. The application also features MVVM architecture.

- Swift
- UIKit
- WebKit
- Search Bar
- TableView
- URLSession
- MVVM
